# Target Price Calculator

A simple web-based calculator.  
Enter values A and B to quickly compute target levels.

Features:
- Two calculation scenarios
- Outputs Target 1 / Target 2
- Works on both mobile and desktop browsers
- No installation required — open and use

# 目标位计算器

一个简单的网页计算器工具。  
输入 A 和 B 数值，可快速计算目标位结果。

功能：
- 支持两种情形计算
- 自动计算目标位一 / 目标位二
- 适配手机和电脑浏览器
- 无需安装软件，打开网页即可使用
